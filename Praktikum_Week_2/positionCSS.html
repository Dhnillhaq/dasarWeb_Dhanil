Position: Static

Position static digunakan untuk mengatur element menjadi statis secara default. Elemen akan mengikuti posisi normal secara default, elemen tidak dipengaruhi oleh properti seperti top, bottom, left, and right.

Elemen ini diatur menggunakan posisi Static




Position: Relative

Sebuah elemen dengan position relative dapat berpindah dan konten lain tidak akan disesuaikan agar sesuai dengan celah yang ditinggalkan oleh elemen tersebut.

Elemen ini diatur menggunakan posisi: relative

Element ini diatur posisi Fixed

Element absolute akan diposisikan relatif dengan element lain yang mendahuluinya yang terdekat, bukan relatif terhadap layar secara normal
